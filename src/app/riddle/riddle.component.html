Eure Aufgabe für die nächsten 60 Minuten: Löst die Rätsel, um den Code am Zahlenschloss der Tür zu knacken!
Die Lösung zu jedem Rätsel ist ein Code, den ihr im Feld unten eingeben und prüfen könnt.
Ist der Code für das jeweilige Rätsel richtig erhaltet ihr das nächste Rätsel.
Irgendwann hoffentlich den finalen Code um das Schloss der Tür zu öffnen und zu fliehen.
Um die Rätsel lösen zu können erhält jeder Spieler einen Zettel mit Hinweisen (s. unten).

Pro Spieler gibt es einen Hinweiszettel (bei drei Spielern muss einer zwei nehmen).
Jeder Zettel enthält Hinweise, die euch zusammen mit dem passenden Rätsel zum richtigen Code führen.
